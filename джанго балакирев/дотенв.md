### Работа с python-dotenv в Django

Для изоляции конфиденциальных данных (секретные ключи, пароли БД, токены платёжных систем) используется файл `.env`, который создаётся в корне проекта и добавляется в `.gitignore`.

1. Установка: `pip install python-dotenv`
2. Настройка в `settings.py`:

```
   python
   import os
   from pathlib import Path
   from dotenv import load_data  # или load_dotenv

   load_dotenv()

   SECRET_KEY = os.getenv('SECRET_KEY')
   DEBUG = os.getenv('DEBUG') == 'True'
```

- **`git remote add origin ...`** — подключает твой компьютер к репозиторию на GitHub (дает этой ссылке удобное имя `origin`).
    
- **`git branch -M master`** — принудительно переименовывает текущую ветку в `master` (делает её главной).
    
- **`git push -u origin master`** — отправляет твой код на GitHub и «связывает» ветки, чтобы в будущем ты мог писать просто `git push`.

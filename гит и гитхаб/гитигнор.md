echo "\*.log" > .gitignore

---- это быстрая команда в консоли чтобы создать специальный файл фильтр для гита. символ > это типа стрелочка которая берет текст из кавычек и закидывает внутрь файла .gitignore который создается в папке проекта. а \*.log внутри означает маску для всех файлов с расширением лог, звездочка типа заменяет любое имя. это нужно делать в самом начале проекта чтобы гит вообще не замечал технический мусор и логи которые система сама создает и они не висели в git status и не мешались под руками когда делаешь коммиты. в общем одной командой отсекаем весь хлам чтобы репозиторий оставался чистым и там лежал только твой реальный код а не автогенерируемые отчеты.

1 --- mkdir app/cd app
2 --- python -m django startproject rasul .
3 --- python manage.py startapp main
4 --- в settings регаем main
5 --- в urls(rasul) прописываем path('', include('main.urls')) url идет в url(main)
6 --- в main создает urls(main) и там импортируем from . import views и прописываем 
    path('', views.home, name='home'),
7 --- в views прописываем функцию ```def home(request):
	return render(request, 'main/index.html')```
8 --- создаем папку templates и в ней сразу папку main (там будут html файлы)
9 --- в ссылках html из-за прописанного name в urls(main) мы можем писать просто 
	{% url 'home' %}


**для передачи данных из вьюс в функции надо передать {'title': 'home'} 
а в шаблоне приписать {{title}}


в функцию можно прописать созданный выше словарь


в словарь можно передать список, который можно перебрать циклом for
{% for i in values %}
	\<p>{{ i }}\<p>
{% endfor %}


также можно передать словарь и обращаться к нему как {{ dict.title }}


условия 
{% if title == 'hello' %}
	\<p>{{ title }}\<p>
{% endif %}

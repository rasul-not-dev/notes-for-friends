3. Для шрифтов — будь осторожен с vw
css
/* ❌ Может стать слишком мелким на телефоне */
h1 {
    font-size: 4vw;
}

/* ✅ Лучше через clamp() — ограничиваем минимум и максимум */
h1 {
    font-size: clamp(18px, 4vw, 48px);
}
clamp() — это супер-функция:

Минимум — 18px (на маленьких экранах)

Идеально — 4vw (адаптируется)

Максимум — 48px (на больших экранах)

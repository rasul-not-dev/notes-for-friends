я еще не настолько хорошо понимаю позиционирование, но вот что для себя отметил из важного:

если задать одному элементу какую либо позицию то он будет смотреть выше и поисках родителя с тоже заданной позицией чтобы позиционироваться относительно него


теперь о якорном позиционировании: 

```
.block1 {
    background: red;
    height: 100px;
    width: 100px;
}
.block2 {
    background: green;
    height: 100px;
    width: 100px;
}
```

первый блок это якорь, а второй - элемент который позиционируется относительно него
подробнее про якорное позиционирование --- https://anchoreum.com/#ru



```
.parent {
  position: relative;
}

.child {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```

--- top: 50%; left: 50% — верхний левый угол ребёнка встаёт в центр родителя
--- transform: translate(-50%, -50%) — сдвигаем ребёнка влево на половину его ширины и вверх на половину его высоты

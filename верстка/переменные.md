
так обьявляются переменные для какого-либо класса и его дочерних элементов:

```
.card {
    --padding: 20px;
    padding: var(--padding);
}

.button {
    --padding: 8px 16px;  /* то же имя, другое значение */
    padding: var(--padding);
}
```

а так обьявляютя переменные которыми можно пользоваться во всем коде:

```
:root { 
	--dark: black; 
	--pink: pink }; 
	
p {
	color: var(--black)
}
```

### 2. ДВА ПОДХОДА К ВЕРСТКЕ

> **Резиновая (плавная):** ширину задаем в `%`, `vw`, `fr`. Сайт сжимается как шарик.
> 
> ​
> 
> **Адаптивная (скачками):** используем `@media`, чтобы переключать дизайн на телефонах и компах.

---

### 3. БРЕЙКПОИНТЫ (Точки перелома)

> **Запомни эти 2 цифры на старте:**
> 
> - `до 767px` = Телефон (верстаем по умолчанию).
>     
> - `от 1024px` = Компьютер (включаем в `@media`).
>     
>     ​
>     
>     _(Планшеты просто подхватят одну из версий, не парься!)_
>     

---

### 4. МАГИЧЕСКАЯ ФОРМУЛА ДЛЯ ШРИФТОВ

> Вместо кучи медиа-запросов используй **`clamp()`**. Это ограничитель.

css

font-size: clamp(Минимум, Идеал, Максимум);

> **Шпаргалка для ленивых:**
> 
> - Заголовки: `clamp(24px, 5vw, 48px)`
>     
> - Текст: `clamp(14px, 2vw, 20px)`
>     

---

### 5. ПРОЦЕНТЫ ВМЕСТО ПИКСЕЛЕЙ

> Если в Фигме ширина блока `300px`, а ширина экрана `1440px` → считаем:
> 
> ​
> 
> `(300 / 1440) * 100% = 20.83%`.
> 
> ​
> 
> _(Но лучше просто юзай `flex: 1` или `grid`, чтобы не считать!)_

---

### 6. ГЛАВНЫЙ СЕКРЕТ (Mobile First)

> **Сначала верстаешь телефон!** Потом, когда станет красиво, пишешь `@media (min-width: 1024px)` и правишь только то, что развалилось на широком экране.

---

### 7. ПИКСЕЛИ В ТЕЛЕФОНЕ (Не путай!)

> **375px** в коде — это **логическая ширина**. Физически у телефона 1000+ точек, но браузер их масштабирует, чтобы текст был крупным. Верстаем всегда под **CSS-пиксели** (375px).




| Единица   | Что значит               | Пример                              |
| --------- | ------------------------ | ----------------------------------- |
| `%`       | Процент от родителя      | `width: 50%`                        |
| `vw`      | 1% от ширины ЭКРАНА      | `font-size: 2vw`                    |
| `vh`      | 1% от высоты ЭКРАНА      | `height: 100vh` (на весь экран)     |
| `fr`      | Доля в Grid/Flex         | `grid-template-columns: 1fr 2fr`    |
| `clamp()` | Минимум, идеал, максимум | `font-size: clamp(16px, 2vw, 24px)` |
|           |                          |                                     |



#### 📱 Простое правило без формул:

| Что верстаешь          | Бери `vw` | Ограничь сверху | Ограничь снизу |
| ---------------------- | --------- | --------------- | -------------- |
| **Огромный заголовок** | `5vw`     | `48px`          | `24px`         |
|                        | `3.5vw`   | `36px`          | `20px`         |
| **Обычный текст**      | `2vw`     | `20px`          | `14px`         |
| **Мелкий текст**       | `1.2vw`   | `16px`          | `10px`         |
